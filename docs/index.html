02 Executor Gateway protects you
All external calls go through a secure MCP server. No keys ever reach agents.
03 TokenOS gives you control
Real-time usage, burn forecasting, and budget guardrails across every tool.
04 Full stack on demand
docker compose --profile full up -d brings up Media + Intelligence layers.
